# 经验系统

在 BLBot 的经验系统中，每次[签到](./credit#签到)成功时，将会获得 1 经验。

各等级和所需经验值如下所示：

| 等级  | 所需经验 |
| ----- | -------- |
| `Lv0` | 0        |
| `Lv1` | 1        |
| `Lv2` | 7        |
| `Lv3` | 15       |
| `Lv4` | 120      |
| `Lv5` | 365      |
| `Lv6` | ∞        |

Bot 的大部分指令只需要 `Lv1`（即首次签到后）即可使用。

在本用户手册中，无等级限制及限制 `Lv1` 使用的指令不再特别标明，需要更高等级才能够使用的指令会用 <LvBadge lv=? /> 的标记标出。
